## 🚀 How to Use This Repo

### 1. Clone the repo
```bash
git clone https://github.com/abhii026/java-journey.git
cd java-journey
```

### 2. Navigate to any topic
```bash
cd basics        # or any folder like oops, strings, etc.
```

### 3. Run a Java file
```bash
javac FileName.java     # compile
java FileName           # run
```

### 4. Open in IntelliJ IDEA
- Open IntelliJ → `File` → `Open` → select the `java-journey` folder
- Navigate to any `.java` file and run it directly using the ▶️ button
